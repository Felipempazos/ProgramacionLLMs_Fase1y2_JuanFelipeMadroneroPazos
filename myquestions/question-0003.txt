El Problema:

Una plataforma de streaming desea agrupar usuarios según sus hábitos de escucha para mejorar sus recomendaciones musicales.

Se recibe un DataFrame de pandas con las columnas:
- "usuario_id" (int)
- "tiempo_pop" (float)
- "tiempo_rock" (float)
- "tiempo_reggaeton" (float)

Debes crear una función llamada segmentar_usuarios_musicales(df) que:

1. Seleccione las columnas numéricas de tiempo de escucha.
2. Escale los datos usando StandardScaler.
3. Aplique KMeans con 2 clusters.
4. Asigne a cada usuario un grupo según su comportamiento.

La función debe recibir:
- df: pandas.DataFrame

La función debe retornar:
- dict[int, int] donde:
  - clave → usuario_id
  - valor → cluster asignado (0 o 1)

Aplicación:
Permite segmentar usuarios según sus preferencias musicales para personalizar recomendaciones.
