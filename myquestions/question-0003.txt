El Problema:

Una plataforma de streaming desea agrupar usuarios según sus hábitos de escucha y entender el tipo de comportamiento musical de cada grupo.

Se recibe un DataFrame de pandas con las columnas:
- "usuario_id" (int)
- "tiempo_pop" (float)
- "tiempo_rock" (float)
- "tiempo_reggaeton" (float)

Debes crear una función llamada analizar_perfiles_musicales(df) que:

1. Seleccione las columnas numéricas de tiempo de escucha:
   ["tiempo_pop", "tiempo_rock", "tiempo_reggaeton"]
2. Escale los datos usando StandardScaler.
3. Aplique KMeans con 2 clusters (n_clusters=2, random_state=0, n_init=10).
4. Para cada cluster, calcule el promedio de tiempo de escucha por género.
5. Determine el tipo de perfil del cluster comparando el valor máximo y mínimo de esos promedios:
   - "especializado" → si (promedio_max - promedio_min) > 20
   - "diverso"       → si (promedio_max - promedio_min) <= 20
6. Asigne a cada usuario la etiqueta de su cluster correspondiente.

La función debe recibir:
- df: pandas.DataFrame

La función debe retornar:
- dict[int, str] donde:
  - clave → usuario_id (int)
  - valor → tipo de perfil ("especializado" o "diverso")

Aplicación:
Permite entender el comportamiento musical de los usuarios y mejorar sistemas de recomendación personalizados.
